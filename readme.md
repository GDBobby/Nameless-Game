Controls
~~~~~~~~~~~~~~~~~~~~~~

MapMaker
	Left click - build, or select block if pointed at menu
	Right Click - delete (build air), or select block if pointed at menu

WASD - movement
Space - Fireball
Ctrl - attack
U - spawn slime
F11 - display character hitbox
